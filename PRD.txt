
电子六爻（数字易经）
小程序 / APP 产品需求文档

PRD v1.3  ·  民俗文化 / 心理健康  ·  待原型设计



本文件为保密资料，仅供产品、设计、技术团队内部使用


1. 产品概述
项目	说明
产品名称	电子六爻（数字易经）
产品类型	微信小程序 + iOS / Android APP
产品定位	以「民俗文化 + 心理疏导」为切入点的易经互动体验工具
核心差异	2.5D 沉浸式起卦动画 + AI 大师姐断卦 + 国学知识卡片
合规定位	心理健康辅助 / 民俗文化研究，非占卜预测类工具

2. 用户旅程总览
主流程：
•首屏  →  选择事项分类  →  补充问题描述  →  2.5D 起卦动画
•排卦结果页  →  AI 大师姐解读  →  分享 / 收藏

「每日一卦」入口常驻首屏，作为独立国学知识内容模块，与主起卦流程并行。

3. 功能模块详述
3.1  起卦前置：事项选择（Input Filtering）
分类选项（单选，必填）
•学业功名（考试、升学、面试）
•事业前程（转岗、创业、合作）
•情缘发展（单身寻找、感情修补、相处之道）
•财运经营（投资、生意、收支）
•寻物杂项（失物、日常疑惑）

规格细节
规格项	说明
输入框	分类选定后可选填具体问题，示例提示：「例：下周二的面试是否顺利」
黑名单过滤	后台配置敏感词（政治、寿命、违法、彩票数字等），命中时提示：「此事项超出了民俗研究范围」
必填校验	未选分类则「起卦」按钮置灰不可点

3.2  2.5D 起卦交互与排卦（Core Interaction）
规格项	说明
视觉方案	2.5D 序列帧动画：铜钱在龟壳（或陶瓷盘）内旋转、跳动、落定
动爻特效	变爻（老阴/老阳）爻位在结果页呈现「呼吸闪烁」或「流光特效」
变卦展示	动爻闪烁后，平移/淡入出变卦爻象，直观呈现「事情在变」易理
频率限制	同一用户 + 同一分类事项 → 24 小时内仅限起卦 1 次
重复提示	「心诚则灵，姐姐建议你先静心感悟今日这一卦，明日再来探寻新变化。」

动画序列帧要求（供设计师参考）
•帧率：建议 ≥ 24fps，序列帧总时长 2.5–3.5 秒
•落定反馈：铜钱落定时配合轻微震动（Haptic Feedback）
•声音：铜钱碰撞音效 + 可关闭的背景音乐，默认静音

3.3  离线 + AI 双层断卦体系（Analysis Logic）
层级	数据来源	输出内容	网络要求
离线基础层	本地内置 64 卦数据库	卦名、吉凶等级（大吉/小吉/平/忧/大忧）、核心一句话大意	无需网络
在线 AI 层	DeepSeek / GPT API	【当下现状】【姐姐建议】【避坑指南】三模块，250–400 字	需联网

AI 大师姐角色设定
设定项	说明
人设	温婉博学、语气舒缓，以心理疏导和国学智慧为核心
禁止行为	恐吓性语言、迷信绝对化预言、确定性的吉凶断语
输出结构	【当下现状】→【姐姐建议】→【避坑指南】，分段清晰
回复长度	250–400 字

AI Prompt 模板（开发者参考）
你是一位博学温婉的易经民俗顾问。请根据用户起的卦象：{卦名}，动爻：{动爻位置}，所求事项：{分类+具体描述}。按照【现状】、【建议】、【避坑指南】三个板块给出一份解读。语气要柔和，严禁出现恐吓、迷信、绝对化预言。请侧重于心理疏导和国学智慧的分享。

3.4  每日一卦：国学知识卡片模块
规格项	说明
入口位置	首屏常驻入口，每日凌晨 00:00 刷新
展示方式	点击后翻开精美民俗卡片（翻牌动效）
内容结构	① 卦名与卦画（如：离为火 ☲☲）② 《易传》/《象传》原文金句 ③ 现代白话解读 ④ 关联历史人物或民间故事
分享功能	一键生成精美海报，含「传承国学文化」角标，规避算命嫌疑
卡片总数	64 张（对应 64 卦），循环不重复

4. 技术规格
模块	技术要求
离线包体	2.5D 序列帧 + 64 卦本地数据库 + 背景音乐全部打包，确保弱网下完整流程可用
AI 接口	接入 DeepSeek 或 OpenAI GPT；接口超时 ≤ 8s；按 §3.3 模板配置 Prompt
敏感词过滤	后台可热更新黑名单，前端触发提示文案（无需重新发版）
频率缓存	本地缓存（小程序 Storage / AsyncStorage）记录「分类 + 时间戳」，24 小时限频
数据埋点	埋点节点：分类选择、起卦成功、AI 解读展示、分享行为

5. 页面清单（原型设计范围）
编号	页面名称	关键交互元素	优先级
P01	首屏 / 首页	「起卦」入口、「每日一卦」入口、历史记录入口	P0
P02	事项选择页	五选一分类、问题补充输入框、确认按钮	P0
P03	起卦动画页	2.5D 序列帧动画、进度指示（第 N 爻）	P0
P04	排卦结果页	六爻排列、动爻闪烁特效、变卦展示、吉凶等级	P0
P05	AI 解读页	三板块解读文本、Streaming 打字效果、收藏/分享	P0
P06	每日一卦页	卡片翻牌动效、内容四段结构、生成海报按钮	P1
P07	历史记录页	时间轴列表、按分类筛选、点击回顾详情	P1
P08	频率限制弹窗	提示文案、倒计时、关闭按钮	P0
P09	分享海报预览页	海报预览、下载/分享至微信	P1

6. 非功能性需求
类别	要求
性能	起卦动画首帧 ≤ 1.5s；AI 解读首字符 ≤ 3s（弱网降级展示离线层）
兼容性	iOS 14+  /  Android 9+  /  微信 8.0+ 小程序
无障碍	核心文案支持系统字体缩放；AI 解读可选开启文字转语音
隐私	用户问题描述不得明文存储上传，仅用于单次 AI 推理
