# 🍃 春风UI升级项目 - 升级说明

## 主题：遇事不决可问春风
为了深化应用的主题，添加了"春风绿叶"的视觉元素和古风配色。

---

## 🎨 核心颜色方案

### 新增春风绿色系
- **主绿色** (#52B788)：用于按钮、高亮、边框
- **浅绿** (#D8F3DC)：背景层，低对比度装饰
- **深绿** (#1B4332)：深层文字，暗场景

### 古风背景色
- **保留** #fcf9f2 (米色)：符合中国古风，已保持不变

## 🎯 设计遵循原则

1. **古风保留**: 保持 #fcf9f2 背景、中文书法字体、竖排文字等古风元素
2. **春风主题**: 绿色代表生机、春天、希望
3. **和谐搭配**: 绿色与米色古风搭配，不突兀
4. **视觉清晰**: 移除扭曲、改进对比度、添加微动画
5. **交互友好**: hover、focus、disabled 状态都有视觉反馈

## 🚀 技术实现

### CSS 文件修改 (`src/index.css`)
```css
/* 新增的春风主题样式 */
- .spring-leaf 类
- .leaf-decoration 类及变体
- sway 动画关键帧
- .spring-green, .spring-accent, .spring-bg-light 等实用类
```

### 颜色替换清单
| 原颜色 | 新颜色 | 用途 |
|------|------|------|
| `text-secondary` | `text-[#52B788]` | 文字颜色 |
| `border-secondary` | `border-[#52B788]` | 边框颜色 |
| `bg-secondary` | `bg-[#52B788]` | 背景颜色 |
| - | `bg-[#D8F3DC]/30` | 卡片浅绿背景 |
| - | `hover:border-[#52B788]` | hover边框 |
